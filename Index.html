Juego del Gato

Introducción

En este código se implementa el juego del gato utilizando métodos que cumplen las funciones correctamente.

Diagrama UML

Estructura

El juego está organizado dentro de una clase llamada Gato. La clase contiene métodos para crear el tablero, manejar las jugadas, verificar si hay un ganador, empate o perdedor y reiniciar el juego cuando termine una partida.

Métodos del Programa

  • __init__(self, raiz): Constructor de la clase que inicializa los elementos principales del juego.
  • crear_botones(self): Crea los botones en la interfaz gráfica de usuario (GUI).
  • realizar_jugada(self, posicion): Ejecuta cada vez que un jugador hace una jugada.
  • verificar_ganador(self): Revisa si alguno de los jugadores ha ganado.
  • reiniciar_juego(self): Reinicia el juego, limpiando el tablero.

Cómo se Juega

El juego se inicia al ejecutar el programa, mostrando una cuadrícula de 3x3 de botones. Los jugadores alternan turnos hasta que uno de ellos coloque tres símbolos iguales en una línea, o hasta que todas las casillas se llenen, resultando en un empate.

Funciones

Las funciones clave del juego son:

  • realizar_jugada: Controla el flujo principal del juego.
  • verificar_ganador: Revisa las combinaciones ganadoras.
  • reiniciar_juego: Restablece el tablero y los turnos.
